# omac wm

Configure the keyboard-driven desktop (AeroSpace + JankyBorders) and apply macOS tweaks.

```
omac wm install   deploy config, apply tweaks, guided first-run
omac wm reload    reload AeroSpace + borders config
omac wm status    show which components are deployed
```

See [Window Management](../window-management/index.md) for the full hotkey map and the focus border.
